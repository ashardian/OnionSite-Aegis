Core System Capabilities

Architect Edition v9.0 modular components.

Neural Sentry

Active Defense Daemon. Continuously monitors circuit creation rates. Triggers NEWNYM signals to sever circuits upon detecting DDoS attacks.

SSH Safety Valve

Cloud-Aware Ingress Logic. A v9.0 exclusive feature that intelligently modifies the NFTables ruleset to permit remote management (Port 22) without compromising the Tor circuit integrity. Critical for VPS deployments.

Balanced Firewall

State-Aware Filtering. Allows "Established" Tor connections while blocking port scans. Includes SSH Safety Valve for Cloud VPS access.

Amnesic Logging

Volatile Memory Architecture. Forces all operational logs into a tmpfs RAM disk. A system reboot results in unrecoverable destruction of evidence.

Containerization

Isolated Runtime. Leverages Seccomp profiles and Capability restrictions to sandbox the application layer. Ideal for high-security deployments.

Traffic Obfuscator

Dynamic Response Manipulation. Implements random timing delays (10-50ms) and response padding to defeat side-channel correlation.

Anti-Forensic Details

Response Manipulation

  • Padding: Responses padded to random block sizes.
  • Timing: 10-50ms jitter added to defeat timing analysis.
  • Headers: Scrubbed of all identifiers (ETag, Server).

Amnesic Storage

  • Logs: Forced to /mnt/ram_logs (tmpfs).
  • Reboot: Data physically destroyed on power loss.
  • Swap: Disabled to prevent leakage to disk.

Frequently Asked Questions

Does this work on Mobile (Termux/NetHunter)?

Yes. The "Bare Metal" installer is optimized for Debian 11+ architectures. You can run a military-grade hidden service directly from a Debian Linux environment on mobile devices using Termux or NetHunter.

Can I use this on a Cloud VPS?

Yes. The v9.0 SSH Safety Valve allows you to whitelist Port 22 during installation, preventing firewall lockouts on remote servers while maintaining Tor anonymity.

Is it really log-free?

Yes. All logs are forced into tmpfs (RAM). Rebooting physically destroys the data. This amnesic storage system ensures zero forensic trace on disk.

What if I get a "Onion Site Not Found" error?

This is normal for Tor v3 addresses during the first 2-5 minutes as they propagate to the directory. v9.0 includes a Balanced Firewall to ensure this propagation is not blocked. Wait a few minutes and try again.
